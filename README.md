# DivsHabbitTracker

A gamified habit tracker. The React frontend (mock data, no backend
required) lives in [`frontend/`](frontend/README.md) — see that README for
setup and full details. `backend/` is an unused placeholder and is not
called by the frontend.

## Quick start

```bash
cd frontend
npm install
npm run dev
```

## Deploying to Vercel

Deploy from this repo root with default settings — the root
[`vercel.json`](vercel.json) builds the `frontend/` subfolder automatically.
See [`frontend/README.md`](frontend/README.md#deployment-vercel) for details
and troubleshooting (including the `404: NOT_FOUND` case caused by Vercel
building from the wrong directory).
